# Cyberpunk Task Manager ⚡

A futuristic, high-performance task management application built with a focus on data persistence and clean logic.

### Key Features:
* **Persistent Storage:** Uses `LocalStorage` so your tasks are saved even after browser refresh.
* **CRUD Operations:** Users can Create, Read, Update, and Delete tasks seamlessly.
* **Advanced Filtering:** Separate views for "All," "Pending," and "Completed" tasks.
* **Neon UI:** Features SVG-animated borders and custom CSS variables for a cyberpunk aesthetic.

### Tech Stack:
* HTML5, CSS3 (Grid/Flexbox), JavaScript (DOM Manipulation).
