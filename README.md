# Task Manager App

## Introduction
The Task Manager App is a simple and efficient way to manage your daily tasks. With features like task addition, editing, completion tracking, and filters, it helps you stay organized and productive.

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks by completion status
- Data persistence using local storage

## Technologies Used
- HTML
- CSS
- JavaScript

## Getting Started
To get a local copy up and running, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/rvanthana02-design/task-manager.git
   ```

2. Open the `index.html` file in your browser to run the app.
